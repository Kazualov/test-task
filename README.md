# Календарь встреч
Легковесный API для планирования встреч, написанный на FastAPI + SQLite.

## Как запустить:
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите сервер: `uvicorn main:app --reload`
3. Откройте Swagger UI в браузере для тестирования: `http://127.0.0.1:8000/docs`
